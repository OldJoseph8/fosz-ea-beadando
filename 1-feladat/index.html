Web-programozás-1 Házi feladat – Projektleírás (összefoglaló) A weboldal egy több oldalas, egységes kinézetű honlap, amely vízszintes menüvel, azonos szerkezeti felépítéssel (header, nav, aside, content, footer) készült. Az alábbi funkciók és oldalak kerültek kialakításra: Index oldal: Egységes fejléc és lábléc, vízszintes menü, aktív menüpont kiemelve. Táblázat oldal: 4x4-es dinamikus táblázat CRUD műveletekkel, rendezéssel, kereséssel, mezőellenőrzéssel. HTML5 funkciók: Web Storage, Workers, Geolocation API stb. – mindegyikhez külön bemutató példa. ChartJS oldal: 5x5-ös táblázatból kiválasztható sor adatai vonaldiagramként megjelenítve Chart.js segítségével. AJAX oldal: Külső API-t használó AJAX alkalmazás CRUD funkciókkal, JavaScript validációval és visszajelzéssel. OOJS oldal: Objektumorientált JavaScript példa class, extends, super stb. elemekkel. React SPA: Két különálló React alkalmazás egy oldalon belül (például játék vagy kalkulátor), komponensekkel, useState használattal.
